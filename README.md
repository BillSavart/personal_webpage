# Bill Wang Portfolio

A React + TypeScript + Bootstrap portfolio site designed for GitHub Pages.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The Vite base path is configured as `/personal_webpage/` for deployment at:

```text
https://BillSavart.github.io/personal_webpage/
```

GitHub Pages deployment is handled by `.github/workflows/deploy.yml` whenever
changes are pushed to `main`.
